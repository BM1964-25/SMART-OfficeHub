SMART OfficeHub

SMART OfficeHub ist eine lokale Gmail- und Google-Calendar-Arbeitszentrale für Mails, Termine, Dokumentanhänge und Gmail-Entwürfe.

Diese GitHub-Pages-Seite zeigt die Projektinformationen. Die produktive App benötigt einen lokalen Node.js-Server, weil Google OAuth, Gmail API, Calendar API und Gmail-Draft-Updates serverseitige Routen verwenden.

Lokaler Start im Projektordner:

npm start und anschließend http://localhost:8791 öffnen.

GitHub Repository öffnen Statische UI-Dateien ansehen
